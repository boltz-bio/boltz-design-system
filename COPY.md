## Tone

Grounded, technical, confident. **Precision over enthusiasm.**

The brand earns trust by sounding like it knows what it's talking about — not by selling. Headlines state capabilities. Body copy quantifies. CTAs invite action without urgency.

## Language

English only. American spelling (color, not colour).

## Voice principles

- Lead with capability claims before community claims
- Quantify when possible ("4K+ Stars", "1000X faster")
- Engineer-readable: "production-ready", "high-performance infrastructure", "built to integrate"
- Researcher-readable: "frontier", "benchmark", "state-of-the-art"
- Never hype: avoid "amazing", "incredible", "delightful", "revolutionary"

## Copy patterns

### CTA buttons — always end with ↗

```
Try Boltz Lab ↗
Get early access ↗
Read the paper ↗
Build with the API ↗
Explore the models ↗
```

### Eyebrow labels — always "• Sentence case"

```
• Our models
• Platform
• Community
• Build on Boltz
• News
• Pricing
• Security
```

Format: literal bullet dot character, single space, sentence case (not Title Case, not UPPERCASE).

### Stat chips

Format: `Service [icon] Count+ Noun`

```
GitHub 4K+ Stars
Slack 5K+ Members
PyPI 1M+ Downloads
```

### Section headings

- 1 line preferred, max 2
- Lead with the claim, not the context
- ✅ "Models that match state-of-the-art benchmarks."
- ❌ "We're proud to share our latest models."

## Words to use

frontier · foundation · model · capability · benchmark · integrate · production-ready · modular · high-performance · open · scalable

## Words to avoid

awesome · amazing · delightful · easy · simple · magic · revolutionize · transform (as marketing verb) · seamless · effortless

## Excessive "we" / "our"

The brand uses "we" sparingly. Let the work speak. Replace with capability statements:

- ❌ "We built a platform that..."
- ✅ "A platform built for..."

## Exclamation marks

None. Boltz never raises its voice. Single exception: critical system errors.

## Numbers

Use shorthand consistently:
- 1,000 → 1K
- 1,000,000 → 1M
- 4,500 → 4.5K (round to nearest hundred)
