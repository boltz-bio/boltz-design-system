## Conflicts to resolve

### 1. â†— suffix rendering

- DESIGN.md: typographic character "â†—" inside the button after text

**Decision (current):** `<Button suffix="arrow-text">` is the default. Add `suffix="arrow-icon"` as opt-in for circular icon variant.

### 2. EyebrowLabel icon

- DESIGN.md: literal bullet dot character "â€˘"
- Figma: leaf icon (Iconoir)

**Decision (current):** Default to `icon="leaf"` (matches live Figma). `icon="bullet"` available as opt-in for DESIGN.md-original behavior.

### 3. Typography values

- DESIGN.md says heading-lg is 56px, Figma says 62px
- DESIGN.md uses pixel tracking (-1.0px), Figma uses percent (-4%)

**Decision (current):** Figma wins. Tokens use 62px + `em`-based tracking.
